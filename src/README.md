# Movienation

Search movies by name.

![Home Image](https://github.com/leanug/movienation/blob/main/src/assets/images/screenshot.png)

## Built with:

- [Styled components](https://www.styled-components.com/)
- React
- Context API
- [OMDb API](https://www.omdbapi.com/)

## Install:

	npm install

## Run

	npm start
